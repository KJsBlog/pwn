C++ Priority Queues(优先队列)

C++优先队列类似队列， 但是在这个数据结构中的元素按照一定的断言排列有序。

empty() 如果优先队列为空，则返回真
pop() 删除第一个元素
push() 加入一个元素
size() 返回优先队列中拥有的元素的个数
top() 返回优先队列中有最高优先级的元素
