C++ Stacks（堆栈）

C++ Stack（堆栈） 是一个容器类的改编，为程序员提供了堆栈的全部功能，――也就是说实现了一个先进后出（FILO）的数据结构。

操作 比较和分配堆栈
empty() 堆栈为空则返回真
pop() 移除栈顶元素
push() 在栈顶增加元素
size() 返回栈中元素数目
top() 返回栈顶元素
